# 说明
使用的是uni-app  
ui是uni-ui  
前端框架vue 3  

### 如何启动
npm 
yarn install
npm run dev:h5
npm run dev:mock

HbuilderX
首先下载HbuilderX
打开终端执行yarn install
然后导入本地项目 > 运行浏览器 > Chrome
